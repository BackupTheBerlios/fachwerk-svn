Quick start

Steps

  1. Nodes (new )
    A new node is inserted and its properties window opens, there you can enter the coordinates. The z-coordinate shows downwards.
    After closing the window, the node stays selected. If you want to correct its coordinates, just click the right mouse button to reopen the properties window.


  2. Members (new )
    Left click on the first node and drag the mouse cursor to the second one. The new member stays selected.


  3. Bearings and Loads
    - Select the node (): When the node gets highlighted, click the left mouse button. The node then stays selected.
    - Click the right mouse button to open the node's properties window. There you can enter the bearing conditions and the loads.


  4. Calculate ()
    Members that have been calculated are drawn with thick lines. Green colour indicates compression force and black tensile force.
    Activate the layer to see the arrows that represent the bearing forces. Now move the mouse cursor over a node with bearing. It gets highlighted and you'll see the values of the bearing force in the status line.


  5. Assign member forces
    If the system is statically indeterminate, not all member forces are determinate. These members are drawn with thin lines.
    - Select such a member (): When the member gets highlighted, click the left mouse button. The member then stays selected.
    - Click the right mouse button to open the member's properties window. There you can set the member force, i.e. the yielding force of a number of reinforcement bars.


  6. Calculate ()
    If necessary, repeat step 5.


  7. Save


  8. Print system
    Verify your model (coordinates, loads) on the printing output.


Examples

There is a hard coded example that can be accessed in the menu «help → example». Load it to get familiarized with the program.

You'll find other example files in the directory «fachwerk3d/bsp».
