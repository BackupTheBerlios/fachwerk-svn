Hintergrundzeichnungen

Durch das Darstellen einer Zeichnung im Hintergrund wird ein Fachwerkmodell anschaulicher. Hintergrundzeichnungen haben jedoch noch eine weitere Funktion; sie können die Eingabe des Modells erleichtern. Zur Eingabe neuer Knoten können Punkte, Linienende, Schnittpunkte, etc des Hintergrunds gefangen (snap) werden.



dxf (CAD – Zeichnung)

Zur Erstellung der Zeichnung wird ein CAD-Programm benötigt.

Es werden Linien (Line, Polyline, LwPolyline), Punkte, Kreise und Bögen dargestellt. Ausgelassen werden u.a. Text sowie so genannte Inserts und Solids. Formatierungen wie Strichdicken und -farben werden ignoriert. Die eingelesene Datei bleibt unverändert.



bgd (Textdatei)

Wer über kein CAD-Programm verfügt, kann die Zeichnungen auch mit einem einfachen Texteditor erstellen, und als Datei mit der Endung bgd abspeichern. Beispiel:

# Beispiel für eine mit einem Texteditor erstellte Zeichnung

Punkte

0 6 // x- und z-Koordinaten des Punktes

4 2

4 6

9 2

13 0 2 // Die Eingabe im Format x y z ist erlaubt, in Fachwerk werden nur der x- und der z-Wert eingelesen.

13 6

Linie // Diese Linie verbindet zwei Punkte

9 2

13 2

Linie // Werden mehr als zwei Punkte angegeben, führt die Linie durch alle Punkte, vom ersten bis zum letzten.

0, 6

4; 2

9,2

Kreis

5 5 // Mittelpunkt

2 // Radius

Bogen

13 6 // Mittelpunkt

2 270 0 // Radius, Start-, Endwinkel (im Gegen-Uhrzeigersinn)

Die einzelnen Zahlen auf einer Zeile können durch Leerzeichen, Tabulator oder ein Komma getrennt sein. Kommentare beginnen mit // oder # oder dem Wort rem. Die Reihenfolge der Befehle (Punkt, Linie, ...) ist belanglos, die Datei darf z.B. mit Linien beginnen, anschliessend Punkte definieren und dann wieder eine Linie... Für die Befehle ist i.d.R. nur der erste Buchstabe massgebend, Gross- und Kleinschreibung wird nicht unterschieden.



csv (Tabellenkalkulation)

Eine Liste mit Punktkoordinaten kann in einem Tabellenkalkulationsprogramm erstellt werden und als csv – Datei abgespeichert werden. In Fachwerk lassen sich diese Punkte im Hintergrund einblenden und neue Knoten können durch Anwählen mit der Maus sehr schnell erstellt werden.
