Hintergrundzeichnungen

Durch das Darstellen einer Zeichnung im Hintergrund wird ein Fachwerkmodell anschaulicher. Hintergrundzeichnungen haben jedoch noch eine weitere Funktion; sie können die Eingabe des Modells erleichtern. Zur Eingabe neuer Knoten können Punkte, Linienende, etc. des Hintergrunds gefangen (snap) werden.



dxf (CAD – Zeichnung, 3D)

Zur Erstellung der dreidimensionalen Strich-Zeichnung wird ein CAD-Programm benötigt.

Es werden Linien (Line, Polyline, LwPolyline) und Punkte dargestellt. Ausgelassen werden u.a. Kreise, Bögen, Text, sowie so genannte Inserts und Solids. Formatierungen wie Strichdicken und -farben werden ignoriert. Die eingelesene Datei bleibt unverändert.



bgd (Textdatei)

Wer über kein 3D-fähiges CAD-Programm verfügt, kann die Zeichnungen auch mit einem einfachen Texteditor erstellen, und als Datei mit der Endung bgd abspeichern. Beispiel:

# Beispiel für eine mit einem Texteditor erstellte 3D-Zeichnung

Punkte

0 3 6     // x-, y- und z-Koordinaten des Punktes

4 3 2

4 3 6

9 3 2

13 0 2

13 6     // Die Eingabe im Format x z ist erlaubt, in Fachwerk3D wird der y-Wert zu Null gesetzt.

Linie     // Diese Linie verbindet zwei Punkte

9 4 2

13 4 2

Linie     // Werden mehr als zwei Punkte angegeben, führt die Linie durch alle Punkte, vom ersten bis zum letzten.

0, 4, 6

4; 4; 2

9, 4, 2

Kugel

5 4 5    // Mittelpunkt

2     // Radius


Die einzelnen Zahlen auf einer Zeile können durch Leerzeichen, Tabulator oder ein Komma getrennt sein. Kommentare beginnen mit // oder # oder dem Wort rem. Die Reihenfolge der Befehle (Punkt, Linie, ...) ist belanglos, die Datei darf z.B. mit Linien beginnen, anschliessend Punkte definieren und dann wieder eine Linie... Für die Befehle ist i.d.R. nur der erste Buchstabe massgebend (Ausnahme Kugel), Gross- und Kleinschreibung wird nicht unterschieden.

Das Dateiformat bgd für 2D und 3D Zeichnungen unterscheidet sich nicht. Fachwerk 3D kann jedoch keine Bögen und Kreise, dafür Kugeln darstellen. Am besten schaut man sich die mitgelieferte Beispieldatei an.



csv (Tabellenkalkulation)

Eine Liste mit Punktkoordinaten kann in einem Tabellenkalkulationsprogramm erstellt werden und als csv – Datei abgespeichert werden. In Fachwerk lassen sich diese Punkte im Hintergrund einblenden und neue Knoten können durch Anwählen mit der Maus sehr schnell erstellt werden.
