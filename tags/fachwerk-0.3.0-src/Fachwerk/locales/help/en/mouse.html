Mouse controls

Buttons

left side:

calculate

Starts the calculation.

reset

Resets all calculated member and bearing forces to «indeterminate».

select a node

Left click on the node you want to select.

select a member

Left click on the member you want to select.

right side (layers):

dxf


Shows the background drawing.

Nr


Shows the node numbers.

Nr


Shows the member numbers.

kN


Shows the member forces if they are known. Also labels the force arrows.

kN


Draws the load vectors.

kN


Draws the bearing force vectors if they are known.



top row:

zoom all


zoom out


zoom

Press the left mouse button and drag a window around the region you want to zoom in.

new

new node

A new node is inserted and its properties window opens.
If you click on the abort button in the properties window, the node remains (at 0/0). To remove it, just press the button.

snap

new node, snap

Snaps to background points. A new node is inserted at that place.
If the background drawing is turned off, this function is used to split an existing member in two parts.

new

new member

Press the left mouse button over the first node and release it over the second node.

remove

First select a node or a member. Then press this button to remove the selected object.

move a node

Press the left mouse button over the node you want to move and release it at the new position.

reset all

Resets the calculation and resets all the assigned member forces to indeterminate.

properties

First select a node or a member. Then press this button to open its properties window..



Selecting

There are four different selecting modes:

AUTOMATIC (Info)

When you move the mouse cursor over a node or member, the object gets highlighted and you'll see information about it in the status line. When you move the cursor away, the yellow marking disappears. In this mode you can't select any node or member.

DOING NOTHING


NODE

When you move the mouse cursor over a node, it gets highlighted and you'll see information about it in the status line. Press the left mouse button to select the highlighted node. The marking remains.

MEMBER

When you move the mouse cursor over a member, it gets highlighted and you'll see information about it in the status line. Press the left mouse button to select the highlighted member. The marking remains.



Once a node or a member has been selected, the yellow marking remains, even if you move the mouse cursor away (mode «DOING NOTHING»). Now you can open the object's properties window, just click the right mouse button or .

In order to re-enter the AUTOMATIC mode, left click anywhere in the main window.


Pan

Press the middle mouse button to pan the view of the model. Or hold down the Ctrl-key and drag the view while pressing the left mouse button.


Zoom

The mouse wheel can be used to zoom. For printing use the key <F4> and drag a zoom frame instead.
