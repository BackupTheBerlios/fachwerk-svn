Bug reports

Please report bugs to the author.

Apart the detailed description of the bug the following information is important:

  • When does the error occur?

  • Screenshot if the bug is graphically visible.

  • A fachwerk file, that allows to reproduce the bug.

  • Version of the program

  • Version of java (type «java -version» in a shell)

  • Operating system



Further development

It is planed to maintain the program (bug fixes, small improvements, etc). It may be worth to check the homepage for a new version from time to time.

It is not planned to extend the functionality of the program. «Fachwerk» essentially has the functionality that it should have. There will only be extensions required by the author.

BUT: Fachwerk is subject to the free licence GPL, the source code is distributed together with the program (in the directory «fachwerk/src»)! Everybody is free to improve or extend the program under the terms of the GPL. Or maybe fachwerk just serves as a suggestion for your own development. Please do not forget to let the author know, in order he also can benefit form your development.
