Menu

Most of the menu items are self-explanatory.



Main

print system

The printout contains much information and therefore it is well suited for the documentation of the modelling.
The upper third of the sheet is used for the graphical representation of the truss («zoom all»). The layers 'member forces', 'loads' and 'bearing forces' are being printed. Below all the nodes and members are listed, showing their coordinates, loads and member forces.

print graphic

The graphical representation of the model will be printed. The zoom and layer settings will be respected. You can specifiy the scale.

load

You can find examples in the directory «fachwerk/bsp».

load background

A background drawing in the format dxf, bgd or csv can be read in. When you save your strut-and-tie model, the background drawing will NOT be saved.

language

Choose the application language. The item «system» stands for the default value of your computer. That is actually the preset value when the program starts. The item «...» allows to choose the language by typing its abbreviation: «de» for German, «en» for English or «fr» for French. If other translations are available (i.e. Russian), you can choose it («ru») this way.
If you want to start the program directly in English (but your system's default is not English), call the program the following way: «fachwerk --language en».

If there are several nodes at the same position, it can be useful to select a node by its number.

The node or the member to be removed must be selected previously, otherwise nothing happens. If you remove a node, all the members connected to it will be removed too.

reset all

Resets the calculation and resets all the assigned member forces to indeterminate.

properties

Opens the properties window of the previously selected member or node.

Calculation

Each of the following calculation methods can be activated or deactivated. Normally both static methods are activated.

preconditioning

Often the number of unknown member forces can be reduced by this method.
Additionally the algorithm serves to localize inconsistencies in the nodes equilibrium.
The 'preconditioning' can be turned off safely.

solver

The system of equations contains two equilibrium conditions for each (open) node. The global equilibrium contributes three equations.
This solver has a special quality: it finds determinate member and bearing forces, even if the system as a hole is statically indeterminate.
Usually the 'solver' must be turned on.

mechnisms (kinematic method)

The program looks for mechanisms proving that equilibrium is not satisfied. A mechanism is shown graphically. Mechanisms due to instabilities that do not contradict equilibrium are disregarded.


Additions

This item contains features that extend the basic functions of the program.

transformation of coordinates

Scale or move the coordinates of the nodes. The scaling operation is applied first.

transformation of background

Scales or moves the background drawing. The original dxf-file remains unchanged. Preferably this transformation is done in the program that is used to create the drawing, because any transformation done within Fachwerk will not be saved.

scaling of loads

Scales all the loads with a factor. Member forces that have been set can be scaled too.

find node position

The selected node will be moved along a straight line, until a required member force is achieved.
Example: Find the exact node position in order that the member force in a «stabilizer» really is zero. Afterwards you can remove that member.
First select the node you want to move. You will be asked for a vector. The straight line is actually defined by the initial node position and this vector. If the node contains a load, the vector of the force will be proposed. The vector is also used to define the second starting point of the iteration.
If no node position is found, that leads to the required member force, the initial situation will be restored. (I.e. because the member force is independent of the node position, or because the iteration using the given starting points doesn't converge.)

propose member forces

Proposes member forces if the system is statically indeterminate. The calculation is linear-elastic and assumes a ratio compression to tension stiffness EA,c/EA,t = 10
One has to judge if the proposed member forces are reasonable for the given case.
